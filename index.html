WELCOME TO CODE SQUAD SCHOOL

Code Squad Educational Society laid the foundation of veritable learning in Buxar by establishing Code Squad School in the year 2001 in a collaborative effort with the Code Squad Schools Society, which is one of the largest chains of CBSE schools in India. Code Squad school Buxar is affiliated to CBSE, New Delhi and offers schooling from Nursery to Class XII. The last twenty years have been a journey where perseverance and progress have been the cornerstones. Today it can proudly claim to be one of the most sought after schools in Eastern India.Far from the bustling urban environs, Code Squad school is situated near the Battle ground of Buxar in village Kathkauli. The sylvan and pristine surroundings lend a serene touch to the impressive campus spread over several acres. A marvel of modern architecture, the campus contains well-lit airy classrooms, outdoor theatre, performing art block, cafeteria, spacious courtyards for indoor games, and beautifully landscaped lawns.
